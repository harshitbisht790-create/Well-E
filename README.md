Backend server for mini AI robot well-E
# well-ERobot AI Server

Ye repository **backend server** provide karta hai Rudra mini robot ke liye.

Features:
- Connects ESP32 robot to OpenAI API
- Handles AI requests safely
- Returns AI reply to robot
- Easy to deploy on Render.com (free tier)

**Files Included:**
- server.js : Node.js backend code
- package.json : Dependencies
- .env.example : API key template (Do not push real key)
